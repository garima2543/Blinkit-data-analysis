#  Blinkit Sales Analysis | Power BI & Python

##  Project Overview
This project analyzes Blinkit sales data to understand customer behavior, sales performance, product trends, and delivery insights. Python was used for data cleaning and exploratory data analysis (EDA), and Power BI was used to build an interactive dashboard for business insights.

---

##  Objectives
- Analyze overall sales and revenue trends
- Identify top-performing products and categories
- Study customer purchase behavior
- Evaluate city-wise and time-based sales patterns

---

##  Tools & Technologies
- Python (Pandas, NumPy)
- Power BI
- Power Query
- DAX
- Data Cleaning & EDA
- Data Visualization

---

##  Data Workflow
1. Cleaned and prepared raw sales data using Python
2. Performed EDA to identify trends and anomalies
3. Built interactive Power BI dashboards with slicers and KPIs

---

##  Key Dashboard Features
- Total sales, revenue, and order KPIs
- Category-wise and product-wise sales
- City-wise performance analysis
- Time-based trends (daily / monthly)

---

##  Key Insights
- A small set of product categories contributes to the majority of revenue
- Sales peak during specific time slots and days
- Certain cities show consistently higher order volumes

---

##  Project Deliverables
- Power BI dashboard
- Python EDA scripts/notebooks
- Cleaned dataset
- Dashboard screenshots
